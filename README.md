# Plastics Additives / Grad-CAM++ 1D CNN

This repository contains code, notebooks, and supporting data for Grad-CAM++ analysis on polymer, waste, and mixture datasets.

## Structure
- `polymers/` : polymer-related data and notebooks
- `waste/` : waste-related data and notebooks
- `mixtures/` : mixtures data and notebooks

## Requirements
Python, Jupyter Notebook, and the packages used in the notebooks.

## Notes
Large generated files, checkpoints, and notebook checkpoint folders are ignored via `.gitignore`.
